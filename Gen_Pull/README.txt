1. Place the entire “Gen_Pull” folder onto your desktop.
2. Open the Genome_get.tcl file in your preferred text editor.
3. Edit the organism name as described at the start of the script.
4. Copy the entire contents of script into a tcl shell.

——

The output will be put into an automatically created folder ~/users/(user)/(organism)
The downloaded genomes will be in the bin folder within that.
